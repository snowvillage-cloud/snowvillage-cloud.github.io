# 運営卒業生ページ (`/about/alumni/`)

過去にコミュニティの立ち上げや発展に貢献してくれた元運営メンバー（Alumni）に敬意を表し、紹介するための特設ページです。

## 📄 ファイルの説明

- **`index.html`**
  - 卒業生の一覧ページです。ルートディレクトリから見て2階層深い場所（`../../`）にあるため、CSSや画像へのリンクパス（相対パス）に注意してください。
  - **卒業生を追加する場合:** `<div class="card reveal member-card">` のブロックを追加し、在籍期間や貢献内容を記述してください。

## 🔗 関連リソース
- このページの専用スタイルは `/css/about.css` に記述されています。
- 使用する画像は `/images/alumni/` に保存してください。
