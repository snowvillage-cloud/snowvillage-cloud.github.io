# 🎯 Focus Areas (`/about/departments`)

運営メンバー（SnowVillage Mayors）が管轄する「4つの注力領域（部門）」の詳細な役割と、具体的な業務内容をまとめたページです。

## 📂 ファイル構成と役割

このディレクトリは、運営メンバー紹介ページ（`/about/`）に設置された「部門タグ」からの遷移先（詳細解説ページ）として機能します。

- **`index.html`**:
  4つの部門の詳細を記載したメインのHTMLファイルです。
  各セクションにはID（例: `id="events"`）が付与されており、前ページからのアンカーリンク（ページ内ジャンプ）の着地点となっています。

- **`../../css/departments.css`**:
  このページ専用のスタイルシートです（ルートの `css` フォルダ内で一元管理）。
  リストの「✦」アイコン装飾や、アンカーリンクで飛んできた際に固定ヘッダーの下にコンテンツが隠れないようにする `scroll-margin-top` などの制御を行なっています。

## 💡 運用・更新のポイント

### 業務内容の追加・変更
現在のところ、このページのコンテンツは `index.html` 内に直接（ハードコードで）記述されています。業務内容（`<li>` タグ）の追加や文章の変更がある場合は、直接HTMLを編集してください。

**▼ 編集箇所（`index.html`内）**
```html
<ul class="task-list">
  <li><strong>新しい業務:</strong> ここに説明文を書きます。</li>
</ul>
```

### 🔗 アンカーリンクの仕様について
`/about/index.html` のJS側で、以下のIDに対してリンクが生成される仕様になっています。もし部門名を変更する場合は、双方のIDの整合性に注意してください。

1. `#events` : イベント・エクスペリエンス部門
2. `#content` : コンテンツ・プラットフォーム部門
3. `#enablement` : 技術育成部門
4. `#programs` : プログラム・オペレーション部門
