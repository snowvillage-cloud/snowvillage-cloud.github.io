# 運営・スポンサーページ (`/about/`)

このディレクトリは、SnowVillageの運営メンバー（Organizers）と、協賛企業（Sponsors）を紹介するページを管理しています。

## 📄 ファイルの説明

- **`index.html`**
  - 運営メンバー紹介、スポンサー紹介、および卒業生ページ（`/about/alumni/`）への導線が一つにまとまったページです。
  - **メンバーを追加・変更する場合:** `<div class="card reveal member-card">` のブロックをコピー＆ペーストして修正してください。
  - **スポンサーを追加・変更する場合:** Gold Sponsors または Silver Sponsors のセクションにある `<div class="card reveal sponsor-card">` のブロックを編集してください。

## 🔗 関連リソース
- このページの専用スタイルは `/css/about.css` に記述されています。
- 使用する画像は `/images/organizers/` および `/images/sponsors/` に保存してください。
