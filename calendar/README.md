# 📅 Event Calendar (`/calendar`)

SnowVillageの全体および各分科会のイベントスケジュールを、Googleカレンダーの埋め込み機能を利用して表示するディレクトリです。

## 📄 ファイルと実装ロジックの解説

### `index.html`
自前でカレンダーのUIを構築するのではなく、運用効率を最大化するために **Google Calendarの `iframe` 埋め込み** を採用しています。

- **Google Calendar Embed**: 
  `<div class="calendar-wrapper">` 内に `iframe` を配置しています。運営側がGoogleカレンダー上で予定を追加・変更すると、サイト上にもリアルタイムで反映されます。
- **ユーザー同期機能**: 
  埋め込みカレンダー右下の「+ Google カレンダー」ボタンを通じて、ユーザー自身のカレンダーにSnowVillageの予定を直接同期（購読）してもらうことが可能です。
- **Component Injection**: 
  `renderHeader("..", "calendar")` を呼び出すことで、階層パスを解決しつつ、グローバルナビゲーションの「Calendar」をアクティブ状態にしています。
- **UX導線**: 
  カレンダー下部に `← イベント一覧・TECH PLAYページへ` のリンクを配置し、詳細なイベントの申し込みページへ迷わず遷移できるように設計しています。

## 🎨 スタイルとレスポンシブ対応
Googleカレンダーの `iframe` はそのままではスマートフォンではみ出してしまうことがあるため、`.calendar-wrapper` というクラスで囲み、CSS側でアスペクト比や `width: 100%` を制御してレスポンシブ対応させています。

## 💡 運用上の注意点
- カレンダーに予定を追加する際は、対象のGoogleカレンダーアカウント（`295ef847e514cf67debce604f99909d12925a5d7b023e44a0d13c668a1e47137@group.calendar.google.com`）の権限が必要です。
- イベントの詳細（申し込みURLなど）は、Googleカレンダーの「予定の説明」欄に記載しておくと、ユーザーがカレンダーをクリックした際にスムーズに案内できます。
