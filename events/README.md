# イベント情報ページ (`/events/`)

SnowVillageで開催される各種イベントの情報と、公式のイベント管理プラットフォームである「TECH PLAY」への導線を集約したページです。

## 📄 ファイルの説明

- **`index.html`**
  - 上部にコミュニティ全体の注目イベント、中部にTECH PLAYへの誘導バナー、下部に各分科会（Chapters）の紹介を配置しています。
  - **注意:** 上部の「注目イベント」のカードはHTML内には直接書かれていません。ルートにある `script.js` の `loadEvents()` 関数によって動的にHTMLが生成され、このページの `<div id="event-grid"></div>` に挿入されます。

## 🛠️ イベント情報を更新するには
1. イベントの基本情報を更新・追加したい場合は、ルートディレクトリの **`script.js` 内にある `eventsData` 配列** を編集してください。
2. 分科会（Chapters）の情報を変更する場合は、この `index.html` を直接編集してください。

## 🔗 関連リソース
- このページの専用スタイルは `/css/events.css` に記述されています。
- 動的生成される注目イベントのサムネイル画像は `/images/events/` に保存してください。
