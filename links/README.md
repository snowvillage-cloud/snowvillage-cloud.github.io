# 🔗 Useful Links (`/links`)

SnowVillageメンバーに役立つ、公式ドキュメントやコミュニティの学習リソースを集約したリンク集ページです。

## 🏗 アーキテクチャ

このページはデータとUIが完全に分離された動的レンダリングを採用しています。

- **`linksData.js`**: リンクのカテゴリー分け、タイトル、URL、アイコン等を管理するデータファイル。
- **`index.html`**: データを読み込み、カード型のリンク集（`.link-card`）としてHTMLを自動生成するコンテナ。

## 💡 リンクの追加・更新方法

HTMLを直接編集する必要はありません。`linksData.js` の配列データを編集するだけで、画面に自動反映されます。

**▼ 新しいリンクを追加する例**
```javascript
{
  title: "表示するタイトル",
  url: "https://...",
  desc: "リンク先の説明文",
  icon: "💡" // 絵文字やSVGを指定
}
```

※リンクはすべて外部サイトとして扱われる前提のため、自動的に別タブで開く（`target="_blank"`）仕様になっています。
