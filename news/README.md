# 📢 News Archive System (`/news`)

過去の全てのお知らせ（ニュース）を一覧表示するためのアーカイブ用ディレクトリです。
メンテナンス性を高めるため、データと基本スタイルをトップページと完全に共有する設計になっています。

## 📄 ファイルと実装ロジックの解説

### `index.html`
このファイル単体でアーカイブページの描画を完結させています。

#### 1. スタイル（CSS）の流用と差分上書き
トップページのニュース枠と同じデザインを保つため、`../css/home.css` を読み込んでいます。
その上で、アーカイブページ特有のレイアウト調整（ヘッダー分の余白や、一覧表示用のゆったりとしたパディング）は、`<head>` 内のインライン `<style>` で上書きしています。
- `.archive-container`: ヘッダー被りを防ぐ `padding-top: 150px`
- `.news-wrapper`: トップページより横幅を広げた `max-width: 1000px`
- `.news-item`: 一覧での可読性を高める `padding: 20px 0`

#### 2. JavaScriptによる動的レンダリング
ページ下部のインライン `<script>` (`DOMContentLoaded` イベント内) で、アーカイブ専用の描画処理を行っています。

- **全件ループ出力**: 
  `../newsData.js` の `newsData` 配列を `map()` 関数で全件ループし、`<ul id="news-full-list">` の中に `<li>` 要素として展開しています。
- **リンクパスの自動補正 (`item.link.startsWith("http")`)**: 
  `newsData.js` に記載されたリンクが `events/` などの相対パスだった場合、`/news` 階層から見るとリンク切れになります。これを防ぐため、内部リンクの場合は自動的に `../` を付与してルート階層に戻す処理を入れています。
