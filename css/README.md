# CSSモジュールディレクトリ (`/css/`)

このディレクトリには、サイトのスタイリング（デザイン）を制御するCSSファイルが格納されています。
メンテナンス性を高めるため、用途やページごとにファイルを分割しています。すべてのファイルは、ルートディレクトリの `style.css` 経由で読み込まれます。

## 📄 各ファイルの説明

- **`common.css`**
  - 全ページ共通の土台となるスタイルです。
  - カラーパレット（CSS変数）、フォント設定、ヘッダー（ナビゲーション）、フッター、共通のボタン（`.btn`）、フェードインアニメーション（`.reveal`）などが定義されています。
- **`home.css`**
  - トップページ（ルートの `index.html`）専用のスタイルです。
  - 主にヒーローセクションのデザインと、CSSのみで描画している「雪の降るアニメーション（`@keyframes`）」が記述されています。
- **`about.css`**
  - 運営・スポンサーページ（`/about/`）および卒業生ページ（`/about/alumni/`）専用のスタイルです。
  - メンバーの円形画像、スポンサーロゴのホバーエフェクト（モノクロからカラーへ）、卒業生ページへの誘導ボタン（`.btn-alumni`）のデザインが含まれます。
- **`events.css`**
  - イベントページ（`/events/`）専用のスタイルです。
  - 動的に生成されるイベントカードのレイアウト、TECH PLAYへの目立つ誘導バナー、分科会（Chapters）ごとのカードレイアウトが定義されています。
- **`join.css`**
  - Slack参加ページ（`/join/`）専用のスタイルです。
  - スクロール可能な行動規範のテキストボックス、カスタムデザインのチェックボックス、同意前はグレーアウトされるボタンの制御（`.btn-slack.disabled`）が含まれます。
