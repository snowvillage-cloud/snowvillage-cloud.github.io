# 🖼 Image Asset Management (`/images`)

サイトで使用される全ビジュアル素材（静的画像）を一元管理するディレクトリです。

## 📂 ディレクトリ構成

- **`/organizers/`**: 
  Mayor（現役運営）の顔写真。
  - **推奨サイズ**: 400x400px 以上の正方形
  - **推奨形式**: WebP または JPG/PNG
  - CSS側で `border-radius: 50%` を適用して円形にトリミングされる前提です。

- **`/alumni/`**: 
  Alumni（卒業生）の顔写真
  - **推奨サイズ**: 400x400px 以上の正方形
  - **推奨形式**: WebP または JPG/PNG
  - CSS側で `border-radius: 50%` を適用して円形にトリミングされる前提です。
  
- **`/sponsors/`**: 
  協賛者のロゴ画像。
  - **推奨形式**: 背景透過の PNG または SVG
  - ダークテーマの背景に配置されることを考慮し、白抜き文字や視認性の高いロゴの使用を推奨します。

- **`/events/`**: 
  イベントページの各イベントのヘッダ画像
  - **推奨形式**: 背景透過の PNG または SVG


## 💡 最適化のガイドライン
- 表示パフォーマンス向上のため、写真はアップロード前に圧縮ツール（TinyPNGなど）を通し、1枚あたり 100KB〜200KB 以下に抑えることを推奨します。
- UIアイコンなどの幾何学的な図形は、解像度に依存しない SVG 形式での管理を優先してください。
