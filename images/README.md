# 画像リソースディレクトリ (`/images/`)

Webサイト内で使用するすべての画像ファイルを一元管理するディレクトリです。用途が混ざらないよう、さらにサブディレクトリに分割しています。

## 📁 サブディレクトリの説明と推奨サイズ

- **`/organizers/`** (運営メンバーの画像)
  - 現在のコアメンバーのアバターや写真。
  - 推奨: `1:1` の正方形（例: `400x400px`）。CSSで自動的に正円にトリミングされます。
- **`/alumni/`** (卒業生の画像)
  - 過去の運営メンバーのアバターや写真。
  - 推奨: `organizers` と同様、`1:1` の正方形。
- **`/sponsors/`** (協賛ロゴ)
  - ホバー時に色がつくエフェクトがあるため、カラー画像を用意してください（初期状態はCSSで自動的にモノクロ化されます）。
- **`/events/`** (注目イベントのサムネイル)
  - `/events/` ページで動的生成されるイベントカード上部の画像。
  - 推奨: `16:9` の横長画像。CSSの `object-fit: cover;` により、指定した枠内に綺麗に収まるよう自動調整されます。
