# Slack参加・行動規範ページ (`/join/`)

コミュニティの安全性を保つため、Slackワークスペースに参加する前に「コミュニティ行動規範（Code of Conduct）」に同意してもらうためのゲートウェイページです。

## 📄 ファイルの説明

- **`index.html`**
  - 行動規範の本文と、同意チェックボックス、Slackへの参加ボタンが配置されています。
  - チェックボックスと連動したボタンの有効化/無効化のロジックは、ルートの `script.js` で制御されています。

## 🛠️ 更新が必要な箇所
- **規約の改定:** 本文内の `<div class="rule-item">` を編集して内容を更新してください。
- **Slack招待リンクの更新:** Slackの招待リンク（URL）は定期的に有効期限が切れる場合があります。更新が必要な際は、一番下の `<a id="slack-join-btn" href="ここを新しいURLに変更">` を更新してください。

## 🔗 関連リソース
- このページの専用スタイルは `/css/join.css` に記述されています。
